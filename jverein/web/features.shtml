Features

  • Mitgliedertabelle
    • Anschrift und weitere persönliche Daten
    • Bankverbindung
    • Kommunikationsdaten (Tel./eMail ...)
    • Mitgliedschaft (Eintritt, Austritt, Beitragsgruppe ...)
  • Abbuchung
    • Erstellung einer DTAUS-Datei, die u. a. auch in Hibiscus importiert werden kann
    • Jahresabbuchung für alle Mitglieder
    • Abbuchung für neu eingetretene Mitglieder
    • Zusatzabbuchungen
  • Auswertung der Mitglieder (PDF u. CSV)
  • Statistik
  • Beitragsgruppe
  • Import aus der Sparkassen-Software SPG-Verein
