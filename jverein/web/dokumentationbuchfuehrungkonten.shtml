Auflistung der Konten

Alle verfügbaren Konten werden aufgelistet. Neue Konten können entweder von Hibiscus importiert werden oder manuell neu eingegeben werden. Durch einen Doppelklick auf ein Konto erscheint die Detailansicht. Mit einem Rechtsklick erscheint ein Kontextmenü. Darin kann entweder der Anfangsbestand für das Konto eingegeben werden oder das Konto kann gelöscht werden, sofern keine Buchungen existieren.

Tip: Hier kann auch ein Bargeldkonto eingerichtet werden!

Das Eröffnungsdatum kann bei Konten, die seit Jahren existieren, leer bleiben. Über das Eröffnungsdatum und das Auflösungsdatum wird gesteuert, ob ein Konto im Jahressaldo und im Jahresabschluss erscheint.
