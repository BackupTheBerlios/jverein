Dokumentation: Auswertung: Jubiläen

Die Mitglieder, die ein Mitgliedschaft- oder Alters-Jubiläum begehen, können ausgewertet werden. Unter Stammdaten werden die Jubeljahre eingetragen. Bei der Auswertung kann aus einer Dropdown-Box das Jahr ausgewählt werden. Zur Verfügung stehen das aktuelle Jahr, das Vorjahr und 2 Folgejahre. Die Ausgabe erfolgt im PDF-Format.

Die Auswertung nach Altersjubiläum ist ab Version 1.1 möglich.
