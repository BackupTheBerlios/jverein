Installation

Voraussetzung für die Installation ist ein installiertes Java Runtime Environment (JRE) der Version >= 1.5.

Jameica ist gemäß Anleitung zu installieren.

Anschließend wird Hibiscus gemäß Anleitung installiert.

Für die Installation von JVerein wird das heruntergeladene ZIP-Archiv in das plugins-Verzeichnis von Jameica entpackt. Fertig.

Hinweis! Zur Zeit ist es erforderlich ein aktuelles Nightly-Build von Jameica und Hibiscus einzusetzen. Künftig wird es zu jedem Jameica/Hibiscus-Release auch das passende JVerein-Release geben.
