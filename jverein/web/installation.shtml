Installation

Standardinstallation

Jameica ist gemäß Anleitung zu installieren.

Anschließend wird Hibiscus gemäß Anleitung installiert.

Für die Installation von JVerein wird das heruntergeladene ZIP-Archiv in das plugins-Verzeichnis von Jameica entpackt. Fertig.

Version 1.1 von JVerein setzt die Version 1.7 von Jameica und 1.9 von Hibiscus voraus.

Jameica-Suite für Windows-Benutzer

Marco Hügel hat die Jameica Suite zusammengestellt. Windows-Benutzer können Jameica, Hibiscus, Syntax und JVerein per Setup-Datei installieren. Download der Jameica-Suite

Jameica + Plugins portable

Marco Hügel hat eine Anleitung zur Installation von JRE, Jameica und Plugins auf einem USB-Stick oder anderen Wechseldatenträger zum portablen Einsatz erstellt.

Entwicklerversionen

Entwicklerversionen sind analog zu der oben beschriebenen Art und Weise zu installieren. Allerdings ist das Nightly-Build von Jameica und das Nightly-Build von Hibiscus zu verwenden.

Wichtig! Nie über eine existierende Version kopieren. Immer ein neues Verzeichnis erstellen!

MySQL

JVerein unterstützt seit dem 29.1.2008 auch MySQL. Zur Installation siehe MySQL-Support.
