Administration: Formulare

Ab Version 1.1

In JVerein werden u. a. für Spendenbescheinigungen Formulare hinterlegt. Bei den Formularen handelt es sich um PDF-Dokumente. Zu jedem der Dokumente werden Formularfelder mit ihren Positionen, Zeichensatz und Zeichengröße hinterlegt.

Zunächst erscheint eine Liste der bereits definierten Formulare:

Durch einen Klick auf neu öffnet sich das Formular-Bearbeitungsfenster (siehe unten). Mit einem Doppelklick auf ein Formular öffnet sich das Bearbeitungsfenster (siehe unten). Mit einem Rechtsklick öffnet sich ein Kontextmenü. Damit können die Formularfelder bearbeitet werden, das Formular kann angezeigt werden oder gelöscht werden.

Jedem Formular wird ein eindeutiger Name gegeben. Ein Formulartyp ist auszuwählen. Bei der Neuaufnahme oder beim Update eines Formulares ist eine PDF-Datei auszuwählen.
