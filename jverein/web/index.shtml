Home

JVerein ist eine Open-Source-Vereinsverwaltung mit einer Anbindung an die ebenfalls unter Open-Source-Lizenz stehende Homebankingsoftware Hibiscus.

Die Implementierung erfolgt mit Java. Der Ablauf auf vielen Plattformen ist damit gewährleistet. Als GUI-Framework kommt Jameica zum Einsatz.

Die Mitglieder können mit ihren Namen und Adressen gespeichtert werden. Zusätzlich werden die Bankverbindung, Kommunikationsdaten (Tel./eMail) und Mitgliedschaftsdaten gespeichert. Jedem Mitglied wird eine Beitragsgruppe zugeordnet. Darüber wird der jährliche Beitragseinzug realisiert. Zusätzlich können für jedes Mitglied zu beliebigen Zeitpunkten Zusatzabbuchungen eingegeben werden (Strafgelder, Eigenanteile ...), die ab der Version 0.8 auch wiederkehrend sein können. Für jedes Mitglied können auch Vermerke und Wiedervorlagen erfaßt werden.

Die Abbuchungen werden im DTAUS-Format mit Hilfe meiner Bibliothek OBanToo ausgegeben und können in Hibiscus importiert werden. Für künftige Versionen ist eine direkte Übergabe der Buchungen vorgesehen.

Der Zahlungseingang von Mitgliedern, die nicht am Abbuchungsverfahren teilnehmen kann überwacht werden.

Die Vereinsbuchführung (Einnamen/Ausgaben) wird über die Umsatzkategorisierung von Hibiscus erledigt.

JVerein steht unter GPL. In diesem Rahmen kann JVerein genutzt werden.

Zusätzlich gibt es eine Bitte des Autors: Wenn JVerein "produktiv" genutzt wird, bitte ich um eine kurze Nachricht an heiner@jverein.de mit dem Namen des Vereins und der Anzahl der Mitglieder. Diese Daten dienen der "Erfolgskontrolle".
