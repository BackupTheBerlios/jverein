Rechnung

Ab Version 1.1

Vorbereitend ist sind ein oder mehrere Formulare einzurichten.

Als Formularfelder stehen zur Verfügung:

  • Empfänger
  • Zahlungsgrund 1
  • Zahlungsgrund 2
  • Betrag
  • Zahlungsweg
  • Tagesdatum

Sofern die Abrechnungsdaten von einem Mitglied stammen, stehen auch die Daten der Mitgliedertabelle zur Verfügung

Durch die Abrechnung werden die Informationen in einer separaten Tabelle festgehalten. Daraus können die Rechnungen erstellt werden.

Übersicht über die Rechnungen:

Mit einem Doppelklick auf eine Rechnungszeile öffnet sich das folgende Fenster mit den Rechnungsinformationen. Diese können hier auch bearbeitet werden.

In der Übersicht der Rechnungen können eine oder mehrere Rechnungen markiert werden. Durch einen Rechtsklick öffnet sich ein Kontextmenü. Damit können entweder die Rechnungen ausgestellt werden oder gelöscht werden.
