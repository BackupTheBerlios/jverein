Programmbeschreibung

Mit einem Klick auf Mitglied öffnet sich das Mitlgied suchen Fenster:

Die Mitglieder sind nach dem Anfangsbuchstaben des Namens in die einzelnen Karteireiter einsortiert. Durch einen Doppelklick auf ein Mitglied oder durch Klick auf Neu öffnet sich das folgende Fenster:

Neben den persönlichen Daten (Name, Anschrift, Geburtsdatum) können weitere Informationen gespeichert werden. Die Bankleitzahl und die Kontonummer werden mit Hilfe von HBCI4JAVA auf Gültigkeit geprüft.

Die Beitragsgruppen sind unter Extras | Beitragsgruppe zu erfassen. Unter Austritt wird das satzungsgemäße Ende der Mitgliedschaft vermerkt. Unter Kündigung das Eingangsdatum der Kündigung.

Zusätzliche Zahlungen (z. B. Eigenanteile für Fahrten, Strafgelder ...) können über Zusatzabbuchungen verarbeitet werden. Ein Klick auf Neu öffnet folgendes Fenster:

Abbuchungen werden mit dem folgenden Bildschirm initiiert. Es können

  • Jahresbeiträge
  • Jahresbeiträge für im laufenden Jahr eingetretene Mitglieder
  • Zusatzbeträge
verarbeitet werden.

Wenn Jahresabbuchung angeklickt wurde, werden für alle Mitglieder, die nicht ausgetreten sind, die Beiträge gemäß eingetragener Beitragsgruppe eingezogen. Für Mitglieder, die im Laufe des Jahres eingetreten sind, können ebenfalls die Beiträge eingezogen werden. Dazu wird das Eingabedatum eingetragen, ab dem die Beiträge für nachträglich eingetretene Mitglieder abgebucht werden sollen.

Durch einen Klick auf Zusatzabbuchung werden auch die zusätzlichen Beträge abgebucht.

Die Daten werden in eine DTAUS-Datei geschrieben. Diese Datei kann entweder direkt auf Diskette ausgegeben und bei der Bank eingereicht oder in Hibiscus importiert werden.

Die Zusatzabbuchungen können sowohl bei jedem Mitglied auf dem entsprechenden Reiter angesehen werden, als auch in einer Übersicht für alle Mitglieder. Dabei kann die Anzeige eingeschränkt werden auf Noch nicht ausgeführte Abbuchungen oder nach den verschiedenen Ausführungsterminen. Ist eine Zusatzabbuchung ausgeführt worden, wird intern ein entsprechendes Merkmal gesetzt. Durch einen Rechtsklick auf eine Zusatzabbuchung und die Auswahl Erneut ausführen wird das Merkmal zurückgesetzt.

Der Mitgliederbestand kann nach den unten aufgeführten Merkmalen selektiert werden. Die Ausgabe erfolgt entweder im PDF oder im CSV-Format.Nach der Erzeugung der Datei wird ein entsprechendes Anzeigeprogramm aufgerufen.
