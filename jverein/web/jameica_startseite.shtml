Jameica Startseite

In die Jameica-Startseite kann eine Liste der Wiedervorlagetermine eingeblendet werden. Dazu wird die Startseite von Jameica geöffnet. Unten rechts befindet sich der Knopf Startseite anpassen. Ein Klick darauf öffnet das Konfigurationsfenster. Mit einem Rechtsklick auf JVerein: Wiedervorlage kann die Anzeige aktiviert oder deaktiviert werden.
