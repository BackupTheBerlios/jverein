Übersicht Wiedervorlagen

Für jedes Mitglied können beliebig viele Wiedervorlagetermine hinterlegt werden. Alle Wiedervorlagetermine können auf einem Bildschirm dargestellt werden. Durch einen Rechtsklick können die Wiedervorlagetermin gelöscht werden oder der Erledigungstag eingetragen/gelöscht werden.
